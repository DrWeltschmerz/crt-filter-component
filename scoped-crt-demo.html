How it works

Scoped approach

The apply-barrel-to attribute targets a CSS selector. The barrel distortion SVG filter only renders on that element, leaving the rest of the page flat.

Mode control

Use the controls (top-right) to switch between Behind and On Top modes. Both work independently of the barrel scoping.

Features demonstrated

  • Scoped barrel — Distortion limited to .scoped-target class.
  • Global effects — Scanlines, bloom, fringe cover the entire viewport.
  • Mode switching — Toggle behind/on top behavior from controls.
  • Visual hierarchy — Navigation stays at z-index 1000, always accessible.
