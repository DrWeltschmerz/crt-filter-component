feature-list

Core Effects

  • Scanlines - Shadow-mask scanlines with animated hairlines and phosphor triads (RGB/custom colors)
  • Chromatic Aberration - Color fringing with configurable jitter animation
  • Bloom/Halation - Screen glow with blur and brightness control (capped at 1.5px for performance)
  • Glass Reflection - Optional reflection layer with position/size control
  • Vignette - Edge darkening with customizable radius and feather
  • Film Grain - 60-frame noise pool with configurable opacity
  • Flicker - Random scanline flicker animation (toggleable)

Advanced

  • Barrel Distortion - SVG-based lens curvature (scoped or global)
  • Scanline Masks - 5 types: shadow-mask, aperture-grille, slot-mask, sharp, soft
  • Z-Index Modes - Behind content (z:-1), on top (z:9999), or disabled
  • 21 Presets - Desktop CRTs, arcade, terminal, phosphor colors, professional, specialized
  • Dynamic Controls - Dual slider/number inputs, collapsible sections, preset selector
  • No Dependencies - Works in vanilla JS, React, Vue, Angular, etc.

usage-examples

Minimal (controls only)

<script type="module" src="crt-overlay.js"></script> <crt-overlay controls></crt-overlay>

Default preset (behind content)

<crt-overlay controls scan-opacity="0.85" hairline-opacity="0.18" fringe-opacity="0.25" noise-opacity="0.22" barrel="2" bloom="0.15" flicker mode="1" ></crt-overlay>

On top of everything

<crt-overlay controls mode="2" ></crt-overlay>

Scoped barrel (specific element only)

<crt-overlay controls apply-barrel-to=".monitor-card" barrel="2.4" ></crt-overlay>

JavaScript API

const overlay = document.querySelector('crt-overlay'); // Update settings overlay.updateConfig({ barrel: 2.5, bloom: 0.18 }); // Apply preset overlay.applyPreset('arcade-crt'); // Listen for changes overlay.addEventListener('config-change', (e) => { console.log('Config updated:', e.detail); });

Z-Index Modes (via controls or mode attribute)

mode="0" // Disabled - no rendering mode="1" // Behind content (z-index: -1) - default mode="2" // On top (z-index: 9999)
notes

What you get

  • Overlay visuals rendered via Shadow DOM layers; no page markup changes required.
  • Bloom is handled by the component (no page-side filters needed).
  • Barrel distortion exported as a namespaced CSS var; opt-in per selector.
  • Controls stay clickable regardless of z-index; hide them by omitting controls.
  • Flicker is internal; subscribe to flicker if you want to react.
